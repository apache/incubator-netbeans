A Aplicação de Amostra Assíncrona do Servlet 3.0.
Com frequência, um servlet tem de esperar a resposta de um recurso, tal como um banco de dados ou uma conexão de mensagem. O processamento assíncrono, uma nova funcionalidade do Java EE 6, evita o bloqueio de solicitações de modo que o thread possa voltar e realizar outras operações.
Este exemplo demonstra a utilização do servlet assíncrono.
