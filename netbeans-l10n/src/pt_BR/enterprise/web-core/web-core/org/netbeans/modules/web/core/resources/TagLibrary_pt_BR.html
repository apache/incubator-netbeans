Cria um arquivo Descritor de Biblioteca de Tag (TLD), que pode ser utilizado em arquivos JavaServer Pages (JSP). Um descritor de biblioteca de tag é um documento XML que contém informação sobre uma biblioteca como um todo e sobre tags e arquivos de tag que estão na biblioteca. Para adicionar tags ao arquivo TLD, utilize o assistente do Handler de tag ou abra o editor XML a partir do nó TLD.
