Создает компоненты Enterprise JavaBean (EJB) сущности с персистентностью, управляемой контейнером (CMP), на основе существующей реляционной базы данных. Компоненты сущностей используются для представления объектов, чей срок жизни дольше типичного времени выполнения программы. Этот шаблон создает компонент сущности для каждой выбранной таблицы, дополненной методами поиска, полями и отношениями CMP.
