Cria um esqueleto de web service com base no arquivo WSDL. Web services são componentes de software reutilizáveis que encapsulam semanticamente funcionalidades discretas. Web services são acessíveis sobre protocolos padrões como o SOAP. Os Web services criados por este modelo são implantados e executados de acordo com os padrões JSR-109 e J2EE 1.4.
