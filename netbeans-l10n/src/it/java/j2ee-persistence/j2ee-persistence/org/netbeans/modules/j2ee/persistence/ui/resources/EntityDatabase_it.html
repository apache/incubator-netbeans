Crea classi entità Java Persistence API basate su un database relazionale esistente. Le classi entità sono usate per rappresentare oggetti il cui ciclo di vita dura più a lungo del normale tempo di esecuzione di un programma. Questo modello crea una classe entità per ognuna delle tabelle sezionate, completa di annotazioni per le query con nome, dei campi che rappresentano colonne, e delle relazioni che rappresentano le chiavi esterne.
