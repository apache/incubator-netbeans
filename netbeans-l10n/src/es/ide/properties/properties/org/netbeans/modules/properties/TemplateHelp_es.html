Crea un archivo de paquetes de recursos (.properties) apropiado para aplicaciones internacionalizadas, separando del código todas las cadenas visibles al humano. Los archivos de paquetes de recursos se pueden utilizar para recolectar otro tipo de cadenas, como propiedades para los scripts de Ant. El paquete de recursos sólo contiene una localización, pero puede agregar localizaciones adicionales en el menú contextual de los archivos creados. El paquete puede ser editado en un archivo de texto (con el formato apropiado) para una localización específica o en una tabla que despliegue información para todas las localizaciones.
