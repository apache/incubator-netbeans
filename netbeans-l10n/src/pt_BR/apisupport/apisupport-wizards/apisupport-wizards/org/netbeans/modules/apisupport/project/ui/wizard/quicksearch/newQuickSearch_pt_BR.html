Use este modelo para registrar um SearchProvider com este módulo. A API de Pesquisa Rápida é uma infraestrutura que permite aos clientes do módulo serem conectados à UI de Pesquisa Rápida e fornece os resultados da pesquisa que o usuário final realizou na UI. A UI de Pesquisa Rápida é representada por um pequeno campo de texto na área da barra de ferramentas no lado direito, com uma janela pop-up associada mostrando os resultados da pesquisa.
