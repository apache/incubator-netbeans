Importation d'une application Java existante et son script de construction dans le projet IDE. Dans les projets Free-Form. les commandes IDE utilisent en fait un script Ant que vous pourrez lancer, compiler et debugger votre projet. Toutes les modifications du process de construction doivent être faites dans votre scirpt Ant. Pour faciliter une maintenance du projet à long terme, utilisez plutôt le template Projet Java avec sources existantes.
