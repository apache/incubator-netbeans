Crea un file di insieme delle risorse (Bundle.properties) utilizzabile per l'internazionalizzazione dell'applicazioni attraverso la separazione dal tuo codice di tuttte le stringhe di testo visibili all''utente. I file di insieme risorse (Resource Bundle) possono essere anche utilizzati per raggruppare altri tipi di stringa, come ad esempio le proprietà per gli script Ant. Il Resource Bundle creato contiene solo una località, ma puoi aggiugerne altre dal menù contestuale dei file creati. Il Bundle può essere modificato in un file di testo (formato file di proprietà) per una specifica località o in una tabella che mostra le informazioni per ciascuna località.
