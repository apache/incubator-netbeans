Пример приложения Pet Catalog иллюстрирует использование приложения JavaServer Faces 2.0 и интерфейса API Java Persistence. Также демонстрируется возможность совместного функционирования GlassFish и MySQL.
Учебный курс по Java EE 6 Pet Catalog можно найти по ссылке http://weblogs.java.net/blog/caroljmcdonald/archive/2009/08/java_ee_6_pet_c.html.
