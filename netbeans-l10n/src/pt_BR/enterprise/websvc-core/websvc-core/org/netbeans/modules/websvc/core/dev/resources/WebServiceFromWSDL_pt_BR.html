Cria o esqueleto de um web service SOAP usando o arquivo WSDL. Web services são componentes de software reutilizáveis projetados para suportar interoperação entre máquinas por meio de uma rede. Os web services SOAP são descritos por um arquivo descritor específico (arquivo WSDL) e podem ser acessados com um protocolo SOAP. Os web services criados por esse modelo são implantados na aplicação web ou no módulo EJB de acordo com os padrões Java EE, como JSR-109 (Implementando Web Services Corporativos).
