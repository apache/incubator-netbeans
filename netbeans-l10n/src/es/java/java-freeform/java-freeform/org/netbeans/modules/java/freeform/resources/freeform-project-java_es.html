Importa una aplicación Java existente y sus secuencias de ordenes de compilación en un proyecto del IDE. En proyectos free-form, los comandos del IDE utilizan el archivo de secuencia de ordenes de Ant que usted provea para ejecutar, compilar y depurar su proyecto. Cualquier modificacion del proceso de compilacion debe ser manejado directamente en su archivo de secuencia de oredenes de Ant. Para facilitar la mantención del proyecto a largo plazo, considere usar la plantilla de proyectos Java con fuentes existentes.
