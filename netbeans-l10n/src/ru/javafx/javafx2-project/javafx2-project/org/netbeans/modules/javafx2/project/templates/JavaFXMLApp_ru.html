Создается новое приложение с поддержкой JavaFX FXML в обычном проекте среды IDE. FXML представляет собой основанный на XML язык разметки с поддержкой возможности создания сценариев, предназначенный для конструирования графов объектов Java. Он предоставляет удобную альтернативу созданию таких графов в процедурном коде и идеально подходит для определения пользовательского интерфейса приложения JavaFX, поскольку иерархическая структура документа XML точно соответствует структуре графа сцены JavaFX.
