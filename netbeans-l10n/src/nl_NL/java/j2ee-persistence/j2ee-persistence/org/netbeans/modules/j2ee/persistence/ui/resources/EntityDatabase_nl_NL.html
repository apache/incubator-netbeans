Maakt Java Persistence API entiteit klassen op basis van een bestaande relationele databank. Entiteits klassen worden gebruikt om voorwerpen the representeren wiens levensduur is langer dan een typische programma uitvoering. Dit sjabloon creëert een entiteit klasse voor elke geselecteerde tafel, compleet met benoemde zoekopdracht annotaties, velden die kolommen representeren, en relaties uitheemse sleutels vertegenwoordigen.
