Importa un'applicazione Java esistente ed il suo script di costruzione in un progetto dell'IDE. Nei progetti in forma libera, i comandi dell'IDE usano lo script Ant che fornisci per costruire, compilare ed effettuare il debug del progetto. Qualsiasi modifica al processo di costruzione dele essere gestita direttamente nel tuo script Ant. Per rendere più facile la manutenzione a lungo termine del progetto, valuta se usare il modello Progetto Java con sorgenti esistenti.
