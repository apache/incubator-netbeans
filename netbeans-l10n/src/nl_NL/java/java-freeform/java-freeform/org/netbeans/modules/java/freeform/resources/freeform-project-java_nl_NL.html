Importeert een bestaande Java applicatie en het bouw-script in een IDE project. In vrije-vorm projecten, IDE commandos gebruiken de ANT-script dat je verschaft, om je project uit te voeren, te compileren,en te debuggen. Iedere modificatie van het bouwproces moet direct door je ANT script behandelt worden. Om het onderhoudt op de lange termijn eenvoudig te houden, raden we aan dat je de Java project met bestaande middelen project sjabloon gebruikt.
