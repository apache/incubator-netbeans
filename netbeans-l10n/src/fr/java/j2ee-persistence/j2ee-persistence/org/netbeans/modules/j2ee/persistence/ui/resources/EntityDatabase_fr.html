Crée les classes entité JPA basées sur une base de données relationnelle existante. Les classes entité sont utilisées pour représenter les objets dont le cycle de vie est plus long qu'un programme typique. Ce modèle crée un classe entité pour chaque table sélectionnée, complète avec les annotations de requête nommée, les champs représentant les colonnes, et les relations représentant les clés étrangères.
