Cria uma caixa de diálogo Swing pré-construída com botões Ok e Cancelar. Você pode utilizar este modelo para criar caixas de mensagens e quaisquer outras caixas de diálogo. Dica: Você também pode criar outros modelos de caixas de diálogo. (Por exemplo, você deve querer um modelo com botões Sim, Não, e Cancelar.) Primeiro crie a caixa de diálogo e depois clique com o botão direito no nó do arquivo-fonte na janela Projetos e escolha Salvar como Modelo a partir do menu de contexto.
