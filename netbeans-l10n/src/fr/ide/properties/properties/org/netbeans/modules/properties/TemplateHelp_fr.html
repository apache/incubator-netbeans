Crée un fichier d''ensemble de ressources (.properties) approprié pour l''internationalisation des applications en séparant l''ensemble des chaines de caractères textes de votre code. Fichiers standard de ressource peuvent aussi être utilisés pour recueillir autres types de chaine de caractères tels que les propriétés pour les scripts Ant. L''ensemble de ressources crée contient seulement une seule version, mais vous pouvez ajouter des fichiers de localisation supplémentaires à partir du menu contextuel du fichier crée. Le bundle peut être édité dans un fichier texte (format fichier de propriété) pour un fichier de localisation spécic ou dans une table qui affiche l''information sur tous les fichiers de localisation.
