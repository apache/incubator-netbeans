Use esse modelo para registrar uma Central de Atualização com esse módulo. Um Descritor de atualização (arquivo XML) será criado pela ação Criar NBMs... em um menu de contexto do conjunto de módulos. Você deve tornar este arquivo XML acessível publicamente por meio de algum URL. Quando este módulo com registro na Central de Atualização for instalado, o gerenciador de plug-ins da aplicação examinará automaticamente a Central de Atualização especificada.
