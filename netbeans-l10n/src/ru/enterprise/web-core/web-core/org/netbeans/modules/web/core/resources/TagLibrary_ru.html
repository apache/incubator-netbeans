Создает файл дескриптора библиотеки тегов (TLD), используемый с файлами страниц JavaServer (JSP). Дескриптор библиотеки тегов является документом XML, содержащим информацию о библиотеке в целом и о тегах и файлах тегов, содержащихся в библиотеке. Для добавления тегов к файлу TLD используйте мастер "Обработчик тегов" или откройте редактор XML из узла TLD.
