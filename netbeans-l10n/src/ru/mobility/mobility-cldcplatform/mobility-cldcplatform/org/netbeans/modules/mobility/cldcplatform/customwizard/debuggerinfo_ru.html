Командная строка отладчика

Параметры:
    platformhome - указанная выше домашняя страница платформы;
    device - указанное выше имя устройства;
    classpath - путь к классам платформы определяется на следующем экране мастера;
    jadfile - имя файла JAD;
    jadurl - URL-адрес JAD для выполнения OTA;
    securitydomain - домен безопасности для выполнения OTA;
    debug - это только условие, указывающее режим отладки;
    debugaddress - адрес отладки;
    debugtransport - транспорт отладки;
    debugserver - режим сервера отладки - по умолчанию значение true;
    debugsuspend - спящий режим отладки - по умолчанию значение true;
    cmdoptions - параметры командной строки согласно настройкам проекта;
    / - специальный разделитель каталогов в ОС.

Форматы параметров:
{argument} - простое замещение, так, например, {jadfile} замещается фактическим путем к файлу JAD.
{argument|command line part} - условие; если при нем для аргумента установлено какое-либо значение, то к командной строке добавляется часть командной строки.
Комбинация условия и простого замещения является допустимой. Например, {jadfile|-Xdescriptor:"{jadfile}"} означает, что к командной строке добавляется Xdescriptor:"/myfolder/my.jad", если аргумент jadfile содержит /myfolder/my.jad.
