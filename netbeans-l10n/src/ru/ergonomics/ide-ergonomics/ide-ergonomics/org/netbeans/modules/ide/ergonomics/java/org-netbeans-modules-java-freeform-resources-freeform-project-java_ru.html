Импортирует существующее Java-приложение и его сценарий сборки в проект интегрированной среды. В проектах свободной формы среда использует предоставленный вами скрипт Ant для выполнения, компиляции и отладки вашего проекта. Для любого изменения процесса сборки нужно вручную изменить ваш скрипт Ant. Долговременная поддержка будет легче при использовании шаблона Проект Java с существующим исходным кодом.
