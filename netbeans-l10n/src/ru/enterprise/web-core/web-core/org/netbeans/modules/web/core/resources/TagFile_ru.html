Создает пустой файл тега. Файлы тегов могут использоваться для создания пользовательских тегов JSP. Это инструмент JSP 2.0, позволяющий записывать теги с помощью синтаксиса JSP. Авторам файлов тегов не требуется знать способы написания классов Java. Файлы тегов необходимо хранить в папке /WEB-INF/tags веб-приложения.
